# WhatsApp Business Automation (WAHA)

## Overview

A self-hosted WhatsApp integration using WAHA (WhatsApp HTTP API) — connect WhatsApp to your business for free with n8n automation, custom triggers, and full data control. No third-party costs.

## Features

- **Self-Hosted** — No dependency on third-party chatbot services
- **n8n Integration** — Powerful workflow automation with visual builder
- **Custom Triggers** — Build any automation workflow you need
- **Full Data Control** — All data stays on your servers
- **Zero Recurring Costs** — Completely free after setup
- **AI-Powered Conversations** — Integrated with custom LLM API for intelligent responses
