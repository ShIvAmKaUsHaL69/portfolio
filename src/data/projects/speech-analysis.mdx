# Speech Analysis Tool

## Overview

A real-time video analysis tool built using React.js, Node.js, and Express to detect smile percentage, filler words, and weak words, generating an improved spoken paragraph for better speech clarity.

## Features

- **Smile Detection** — Analyzes video feed to detect smile percentage during speech
- **Filler Word Detection** — Identifies filler words like "um", "uh", "like" in real-time
- **Weak Word Analysis** — Detects weak language patterns and suggests stronger alternatives
- **Improved Paragraph Generation** — Generates an enhanced version of the spoken content
- **Real-time Processing** — All analysis happens in real-time as you speak
