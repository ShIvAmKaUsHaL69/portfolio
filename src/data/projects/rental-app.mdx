# Rental App

## Overview

A cross-platform mobile application that allows users to log in using Google authentication (powered by Appwrite) and rent apartments or properties seamlessly. Built with React Native for a native mobile experience.

## Features

- **Google Authentication** — Secure login powered by Appwrite
- **Property Listings** — Browse available apartments and properties
- **Cross-Platform** — Works on both iOS and Android
- **Seamless UX** — Smooth and intuitive rental experience
- **Search & Filter** — Find properties by location, price, and type
