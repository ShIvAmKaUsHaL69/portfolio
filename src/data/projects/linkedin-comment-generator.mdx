# LinkedIn Comment Generator

## Overview

An intelligent Chrome extension that generates AI-powered comments for LinkedIn posts, helping you engage meaningfully with your network. Simply click on any LinkedIn post and get contextually relevant, professional comments generated instantly.

## Features

- **AI-Powered Comments** — Generates contextually relevant comments using AI
- **Chrome Extension** — Seamless integration with LinkedIn's interface
- **One-Click Generation** — Generate comments with a single click
- **Multiple Tones** — Choose between professional, casual, or enthusiastic tones
- **Network Engagement** — Helps maintain active engagement with your professional network
